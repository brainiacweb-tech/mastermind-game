# Mastermind Game

> Crack the secret 4-color code using Black/White peg feedback within 10 attempts.

---

## Features
- 6 colors (R G B Y O P)
- Black peg = correct color + position
- White peg = correct color, wrong position
- Up to 10 attempts

## Requirements
```
No external dependencies — pure Python 3
```

## Installation
```bash
git clone https://github.com/brainiacweb-tech/mastermind-game
cd mastermind-game
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```
